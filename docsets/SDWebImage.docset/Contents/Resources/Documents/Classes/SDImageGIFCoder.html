SDImageGIFCoder

@interface SDImageGIFCoder
    : NSObject <SDProgressiveImageCoder, SDAnimatedImageCoder>

Built in coder using ImageIO that supports animated GIF encoding/decoding

Note

SDImageIOCoder supports GIF but only as static (will use the 1st frame).

Note

Use SDImageGIFCoder for fully animated GIFs. For UIImageView, it will produce animated UIImage(NSImage on macOS) for rendering. For SDAnimatedImageView, it will use SDAnimatedImage for rendering.

Note

The recommended approach for animated GIFs is using SDAnimatedImage with SDAnimatedImageView. It’s more performant than UIImageView for GIF displaying(especially on memory usage)
